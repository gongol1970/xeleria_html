Interfaz privada de Pia para Planeta Casa.

Flujo de prueba sin WhatsApp/Meta:
1. Ingresar la clave privada.
2. Crear una conversacion con "Nueva prueba" y escribir el primer mensaje del comprador.
3. Continuar el dialogo con "Comprador".
4. Usar "Tomar" para responder como humano y "Liberar bot" para devolver la conversacion a Pia.
5. Usar "Borrar" para eliminar definitivamente la conversacion.

La ventana espia muestra diagnostico estructurado de deteccion, skills y contexto.
No muestra razonamiento interno libre.
