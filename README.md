# XelerIA HTML

Frontend publico y paneles de XelerIA.

## Alcance

- Landing publica para xeleria.com.ar.
- Panel del SaaS.
- Panel admin interno.
- Assets compartidos.

## Estructura inicial

- `index.html`: landing publica.
- `app/`: panel del SaaS.
- `admin/`: panel interno.
- `assets/`: recursos compartidos.

## Backend

El backend principal vive en `xeleria_app`.

Para MVP:

- `xeleria.com.ar`: frontend.
- `api.xeleria.com.ar`: API FastAPI.
