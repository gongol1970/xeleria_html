Modelo simple

    
Reglas SKU: SKU, retiro permitido, lugar de retiro, Flex, Full, Mercado Envíos, Correo/Encomienda, costo de envío, fuente de stock, modo respuesta y notas para IA.

    
Lugares retiro: clave, nombre, dirección, horarios y notas.

    
El sistema normaliza respuestas tipo SI/NO y las notas se usan como contexto comercial, no como instrucciones para el cliente si no corresponden.

  




Control operativo · Fallas y revisiones
Jobs de sincronización de stock en error/revisión. Usá Procesar pendientes para reintentar sin esperar al worker automático.
